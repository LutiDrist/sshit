4. Философия проекта Ключевые принципы Отвечу прямо:
- **Никакой скрытой магии** Каждый шаг протокола должен быть видимым и понятным.
- **Документация является частью кода** Решения по проектированию записываются, а не хранятся в голове.
- **Обучение важнее скорости** Корректность и ясность важнее полноты функционала. 
- **Дневник инженера** Каждый важный шаг фиксируется с обоснованием и компромиссами.
